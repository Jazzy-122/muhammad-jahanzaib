# MJ Portfolio v2

A premium, responsive personal portfolio website for Muhammad Jahanzaib built with HTML5, CSS3, Bootstrap 5, vanilla JavaScript, Font Awesome, AOS, Typed.js, and Particles.js.

## Features
- Premium glassmorphism aesthetic inspired by modern tech and maritime luxury branding
- Responsive multi-section layout with home, about, experience, skills, education, projects, and contact
- Animated hero, typing effect, counters, scroll progress, and back-to-top action
- SEO metadata, manifest, sitemap, and robots support
- Ready to deploy on GitHub Pages by opening the repository root

## Installation
1. Clone or download the repository.
2. Open index.html in a browser, or serve the folder with a simple local server.

## GitHub Pages Deployment
1. Push the repository to GitHub.
2. Open the repository settings.
3. Under Pages, choose the main branch and root folder.
4. The site will be published at your GitHub Pages URL.

## Folder Structure
- index.html
- css/style.css
- js/script.js
- images/
- assets/
- manifest.json
- sitemap.xml
- robots.txt

## Customization
Update the personal details, contact information, and PDF file in the assets folder to match your own profile.
